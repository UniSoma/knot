# Git & Commits

## No AI attribution

Never include AI attribution in commit messages, trailers, or PR descriptions. This applies to commits, amends, and squashes.

Forbidden:

- "Generated with [Claude Code]" / "Generated by ..."
- AI emojis (e.g. robot face) marking authorship
- `Co-Authored-By:` lines that reference Claude, Copilot, or any AI tool/agent

## Commit format

- Imperative mood ("Add foo", not "Added foo" or "Adds foo")
- Summary ≤ 80 characters, no trailing period
- Body wraps at 80 characters
- Body explains *what* and *why*; the diff already shows *how*

## Pre-commit gate

Before committing:

1. Run `bb test` — see [testing.md](testing.md).
2. Run `clj-kondo --lint src test` — see [linting-and-formatting.md](linting-and-formatting.md).
